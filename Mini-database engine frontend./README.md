# Mini Database Engine – Frontend

This repository contains the frontend code for the **Mini Database Engine** project.
The UI design is based on the original Figma prototype.

🔗 Figma Design: https://www.figma.com/design/z3rB5zgcxtvxFYind7SVaG/Mini-database-engine-frontend

---

## 🚀 Getting Started

Follow these steps to run the project locally:

### 1. Install Dependencies

```bash
npm install
```

### 2. Start Development Server

```bash
npm run dev
```

The app will start on a local development server. Open the displayed URL in your browser.
